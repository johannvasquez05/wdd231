Site Plan – GameHub

Site Name

GameHub

I chose this name because it clearly represents the purpose of the site: a centralized hub where users can browse a simple catalog of video games. It is short, easy to remember, and accurately reflects what users can expect from the project.

Site Purpose

The purpose of this website is to display a collection of video games using dynamic data loaded from a JSON file. Users will be able to browse game cards, view details in a modal window, save favorites using local storage, and access a contact page with a functional HTML form. The layout and design focus on responsiveness, usability, and accessibility.

Scenarios

  • “Which popular games are available and what genres do they belong to?”
  • “How can I view more details about a specific game?”
  • “Can I save my favorite games to look at later?”

Color Scheme

Primary Color – Deep Blue (#1e3a8a)

  • Used for headings, accents, borders, and important UI highlights.

Secondary Color – Light Gray (#f3f4f6)

  • Used for page backgrounds and subtle visual separation.

Accent Color – Soft Gray (#e5e7eb)

  • Used behind blocks, card edges, and section borders for clean structure.

Typography

Heading Font: Montserrat (Sans-serif)

  • Used for all headings (h1, h2, h3)
  • Creates a modern, bold, and technology-focused look

Body Font: Roboto (Sans-serif)

  • Used for paragraphs, lists, and general content
  • Highly readable and works well for mobile and desktop layouts
